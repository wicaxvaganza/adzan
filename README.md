# 🕌 Aplikasi Adzan Otomatis

Aplikasi berbasis **PHP** untuk membantu pemutaran audio adzan secara terjadwal.

## ✨ Fitur Utama

- 🕒 Pemutaran adzan sesuai jadwal
- 🔊 Dukungan beberapa file audio adzan
- 📁 Penyimpanan cache lokal
- 🧾 Logging aktivitas dan error

## 📂 Struktur Singkat

- `index.php` → file utama aplikasi
- `cache_adzan/` → penyimpanan cache
- `*.mp3` → koleksi audio adzan
- `adzan_activity.log` → log aktivitas
- `client_error.log` → log error client

## 🚀 Cara Menjalankan

1. Pindahkan folder proyek ke direktori web server (contoh: `htdocs` di XAMPP).
2. Jalankan Apache melalui XAMPP.
3. Buka browser dan akses:
   - `http://localhost/adzan`
